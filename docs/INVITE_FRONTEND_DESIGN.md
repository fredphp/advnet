### 5.3 佣金明细项组件

```vue
<template>
  <view class="commission-item">
    <view class="header">
      <text class="type">{{ typeText }}</text>
      <text class="status" :class="statusClass">{{ statusText }}</text>
    </view>
    <view class="body">
      <text class="desc">{{ levelText }}下级{{ typeText }}</text>
      <text class="amount">¥{{ commission }}</text>
    </view>
    <view class="footer">
      <text class="time">{{ formatTime }}</text>
      <text class="coin">{{ coinAmount }} 金币</text>
    </view>
  </view>
</template>

<script>
export default {
  props: {
    sourceType: String,      // withdraw/video/red_packet/game
    level: Number,           // 1/2
    commission: String,      // 佣金金额
    coinAmount: String,      // 金币数量
    status: Number,          // 0/1
    createtime: Number       // 时间戳
  },
  computed: {
    typeText() {
      const map = {
        withdraw: '提现分佣',
        video: '视频分佣',
        red_packet: '红包分佣',
        game: '游戏分佣'
      };
      return map[this.sourceType] || '其他';
    },
    levelText() {
      return this.level === 1 ? '一级' : '二级';
    },
    statusText() {
      return this.status === 1 ? '已结算' : '待结算';
    },
    statusClass() {
      return this.status === 1 ? 'success' : 'pending';
    }
  }
}
</script>
```

## 7. 状态码定义

| 状态码 | 说明 |
|--------|------|
| 0 | 待结算 |
| 1 | 已结算 |
| 2 | 已取消 |
| 3 | 已冻结 |

---

## 8. 注意事项

1. **金币换算**: 10000金币 = 1元人民币
2. **分佣延迟**: 提现成功后延迟5分钟发放佣金（可配置）
3. **最大层级**: 当前支持2级，可扩展至3级
4. **有效邀请**: 指已产生过佣金贡献的下级
5. **佣金比例**: 后台可配置，不同来源可设置不同比例
