规则说明

奖励规则说明
  • 奖励规则按周期天数循环生效，例如设置1-7天，第8天签到将获得第1天的奖励
  • 连续签到天数在周期内对应的天数奖励
  • 补签不影响连续签到天数的计算
  • 可自由调整周期天数和对应的金币奖励
补签规则说明
  • 用户可对当月未签到的日期进行补签
  • 补签需消耗金币，补签成功后按对应周期天数发放奖励金币
  • 每月补签次数由"每月可补签天数"控制
