匹配规则：
  1. 系统分别获取时间段配置和今日金额配置的奖励区间
  2. 计算两个区间的交集（取两个区间的重叠部分）
  3. 如果交集有效（下限 ≤ 上限），使用交集区间
  4. 如果交集无效（无重叠），使用时间配置（更严格）

示例：用户今日领取 150,000 金币，当前时间 14:00（12-18点）
  • 时间配置(12-18点)：基础奖励 3000-4000
  • 金额配置(0-20万)：基础奖励 4000-6000
  • 交集结果：[max(3000,4000), min(4000,6000)] = 4000-4000（固定4000）
最高金额限制：红包最高金额为 {$maxRewardLimit|number_format} 金币，不论如何累加都不能超过此限制！
