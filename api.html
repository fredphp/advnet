👤 用户模块

POST /api/user/login 会员登录 公开

用户通过账号密码登录系统，成功后返回用户信息和Token。

请求参数
参数名 类型 必填 说明
account string 必填 账号（用户名/手机号/邮箱）
password string 必填 密码
响应示例
{
    "code": 1,
    "msg": "登录成功",
    "data": {
        "userinfo": {
            "id": 1,
            "username": "testuser",
            "nickname": "测试用户",
            "mobile": "13800138000",
            "avatar": "/uploads/avatar.png",
            "token": "xxxx-xxxx-xxxx-xxxx"
        }
    }
}
POST /api/user/register 会员注册 公开

新用户注册账号，需要手机验证码验证。

请求参数
参数名 类型 必填 说明
username string 必填 用户名
password string 必填 密码（6-30位）
mobile string 必填 手机号
code string 必填 短信验证码
email string 可选 邮箱地址
响应示例
{
    "code": 1,
    "msg": "注册成功",
    "data": {
        "userinfo": {
            "id": 100,
            "username": "newuser",
            "token": "xxxx-xxxx-xxxx-xxxx"
        }
    }
}
POST /api/user/logout 退出登录 需登录

用户退出登录，清除Token。

响应示例
{
    "code": 1,
    "msg": "退出成功"
}
POST /api/user/profile 修改个人资料 需登录

修改用户个人资料信息。

请求参数
参数名 类型 必填 说明
username string 可选 用户名
nickname string 可选 昵称
avatar string 可选 头像URL
bio string 可选 个人简介
POST /api/user/resetpwd 重置密码 公开

通过手机验证码重置密码。

请求参数
参数名 类型 必填 说明
mobile string 必填 手机号
newpassword string 必填 新密码（6-30位）
captcha string 必填 短信验证码
type string 可选 类型：mobile/email，默认mobile

💰 金币模块

GET /api/coin/balance 获取金币余额 需登录

获取当前用户的金币余额。

响应示例
{
    "code": 1,
    "msg": "获取成功",
    "data": {
        "balance": 10000
    }
}
GET /api/coin/account 获取账户详情 需登录

获取用户金币账户详细信息，包括余额、冻结金额、累计收支等。

响应示例
{
    "code": 1,
    "msg": "获取成功",
    "data": {
        "balance": 10000,
        "frozen": 500,
        "total_income": 50000,
        "total_expense": 40500
    }
}
GET /api/coin/logs 金币流水 需登录

获取用户金币流水记录，支持按月份分表查询。

请求参数
参数名 类型 必填 说明
page int 可选 页码，默认1
limit int 可选 每页数量，默认20
type string 可选 流水类型筛选
month string 可选 月份，格式YYYYMM，默认当月
响应示例
{
    "code": 1,
    "msg": "获取成功",
    "data": {
        "total": 100,
        "list": [
            {
                "id": 1,
                "type": "video_watch",
                "amount": 100,
                "balance": 10100,
                "description": "观看视频奖励",
                "createtime": 1678886400
            }
        ]
    }
}

💸 提现模块

GET /api/withdraw/config 获取提现配置 需登录

获取提现相关配置信息，包括汇率、限额、可选金额等。

响应示例
{
    "code": 1,
    "msg": "获取成功",
    "data": {
        "min_withdraw": 1,
        "max_withdraw": 500,
        "exchange_rate": 10000,
        "fee_rate": 0,
        "daily_limit": 3,
        "daily_amount": 500,
        "balance": 10000,
        "frozen": 0,
        "withdraw_amounts": [0.3, 0.5, 1, 5, 10],
        "withdraw_enabled": 1
    }
}
POST /api/withdraw/apply 申请提现 需登录

用户申请提现，支持微信、支付宝、银行卡等方式。

请求参数
参数名 类型 必填 说明
coin_amount int 必填 提现金币数量
withdraw_type string 必填 提现方式：wechat/alipay/bank
withdraw_account string 必填 提现账号
withdraw_name string 必填 收款人姓名
bank_name string 可选 银行名称（银行卡提现时必填）
注意：提现金额必须在系统配置的可选金额范围内，否则会申请失败。
GET /api/withdraw/list 提现记录列表 需登录

获取用户的提现记录列表。

请求参数
参数名 类型 必填 说明
status int 可选 状态筛选：0待审核/1已通过/2已拒绝/3已打款
page int 可选 页码，默认1
limit int 可选 每页数量，默认20

🧧 红包模块

POST /api/redpacket/click 点击红包 需登录

点击红包生成/累加红包金额，支持重置模式。

请求参数
参数名 类型 必填 说明
task_id int 可选 任务ID
reset int 可选 是否重置：1=重置并生成新红包，0=正常累加
响应示例
{
    "code": 1,
    "msg": "获取成功",
    "data": {
        "total_amount": 150
    }
}
POST /api/redpacket/claim 领取红包 需登录

观看广告后领取红包金币。

请求参数
参数名 类型 必填 说明
task_id int 可选 任务ID
响应示例
{
    "code": 1,
    "msg": "领取成功",
    "data": {
        "amount": 150,
        "balance": 10150
    }
}

🎬 视频收益模块

POST /api/video_reward/watch 上报观看进度 需登录

上报用户观看视频的进度数据。

请求参数
参数名 类型 必填 说明
video_id int 必填 视频ID
watch_duration int 必填 本次观看时长（秒）
watch_progress int 必填 观看进度（0-100）
current_position int 可选 当前播放位置（秒）
session_id string 可选 会话ID
POST /api/video_reward/claim 领取视频奖励 需登录

观看完成后领取金币奖励。

请求参数
参数名 类型 必填 说明
video_id int 必填 视频ID
响应示例
{
    "code": 1,
    "msg": "领取成功",
    "data": {
        "reward_coin": 100,
        "balance": 10100
    }
}

👥 邀请分佣模块

POST /api/invite/bind 绑定邀请关系 需登录

新用户绑定邀请人关系。

请求参数
参数名 类型 必填 说明
invite_code string 必填 邀请码
channel string 可选 渠道来源，默认link
GET /api/invite/overview 获取邀请统计 需登录

获取邀请统计概览数据，包括邀请人数、佣金总额等。

响应示例
{
    "code": 1,
    "msg": "获取成功",
    "data": {
        "total_invite": 50,
        "level1_count": 30,
        "level2_count": 20,
        "total_commission": 5000,
        "level1_commission": 3500,
        "level2_commission": 1500
    }
}
GET /api/invite/list 获取邀请列表 需登录

获取邀请的下级用户列表。

请求参数
参数名 类型 必填 说明
level int 可选 层级：1=一级，2=二级，0=全部
page int 可选 页码，默认1
limit int 可选 每页数量，默认20

📬 消息模块

GET /api/message/list 获取消息列表 需登录

获取用户的消息列表。

请求参数
参数名 类型 必填 说明
page int 可选 页码，默认1
pagesize int 可选 每页数量，默认20
type string 可选 消息类型筛选
GET /api/message/unreadCount 获取未读消息数量 需登录

获取用户未读消息的数量。

响应示例
{
    "code": 1,
    "msg": "获取成功",
    "data": {
        "unread_count": 5,
        "total_count": 20
    }
}
POST /api/message/markRead 标记消息已读 需登录

标记消息为已读状态。

请求参数
参数名 类型 必填 说明
message_id int 必填 消息ID，0表示标记全部已读

⚙️ 公共接口

POST /api/common/init 初始化配置 公开

获取应用初始化配置信息，包括上传配置、版本信息等。

请求参数
参数名 类型 必填 说明
version string 可选 客户端版本号
lng string 可选 经度
lat string 可选 纬度
POST /api/common/upload 文件上传 需登录

上传文件到服务器，支持普通上传和分片上传。

请求参数
参数名 类型 必填 说明
file file 必填 文件流
chunkid string 可选 分片上传ID
chunkindex int 可选 分片索引
chunkcount int 可选 总分片数
响应示例
{
    "code": 1,
    "msg": "上传成功",
    "data": {
        "url": "/uploads/2023/03/15/xxx.jpg",
        "fullurl": "https://example.com/uploads/2023/03/15/xxx.jpg"
    }
}

📱 短信验证模块

POST /api/sms/send 发送验证码 公开

发送短信验证码，支持多种事件类型。

请求参数
参数名 类型 必填 说明
mobile string 必填 手机号
event string 必填 事件：register/mobilelogin/changemobile/resetpwd
限制：同一手机号60秒内只能发送一次，同一IP每小时最多发送5次。
POST /api/sms/check 验证验证码 公开

验证短信验证码是否正确。

请求参数
参数名 类型 必填 说明
mobile string 必填 手机号
event string 必填 事件名称
captcha string 必填 验证码

📋 通用说明

请求头说明

参数名 类型 说明
token string 用户登录Token，通过Header或URL参数传递
X-Device-Id string 设备唯一标识，用于风控检测
Content-Type string application/x-www-form-urlencoded 或 application/json

响应格式

字段 类型 说明
code int 状态码：1=成功，0=失败
msg string 提示信息
data object 返回数据
time int 服务器时间戳

错误码说明

错误码 说明
-1 未登录或Token失效
0 请求失败，具体原因见msg
1 请求成功
